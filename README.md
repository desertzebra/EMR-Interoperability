# EMR-Interoperability

Satti, F.A., Hussain, M., Hussain, J., Ali, S.I., Ali, T., Bilal, H.S.M., Chung, T. and Lee, S., 2021. Unsupervised Semantic Mapping for Healthcare Data Storage Schema. IEEE Access.
